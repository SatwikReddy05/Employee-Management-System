# Employee-Management-System


A simple SQL-based project to manage employee departments, salaries, and bonuses using relational database concepts.

## Tech Stack
- SQL (MySQL / SQL Server)

## Key Concepts Used
- Primary & Foreign Keys  
- INNER JOIN  
- Aggregate Functions (AVG, SUM)  
- GROUP BY & ORDER BY  
- Stored Procedures  

## Features
- Average salary by department  
- Highest salary by job role  
- Average and highest bonus analysis
  
## Approach of Queries
 •	Identified primary and foreign keys (Job_ID) to establish relationships between tables.
 
 •	Used INNER JOIN to combine job/department data with salary and bonus details.
 
 •	Applied aggregate functions (AVG, SUM, MAX, COUNT) to analyze salary and bonus trends.
 
 •	Grouped data using GROUP BY to perform department-wise and role-wise analysis.
 
 •	Sorted results using ORDER BY to identify highest, lowest, and top-performing departments or roles.
 
 •	Limited results (LIMIT) to extract top or bottom records for decision-making.
 
 •	Designed queries to answer real business questions like cost distribution and compensation insights.
